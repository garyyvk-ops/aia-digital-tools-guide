AIA DIGITAL TOOLS GUIDE - INDEPENDENT WEB EDITION

This is a normal static website. It does not require Articulate, an LMS,
an account, or a learner login.

TO PUBLISH
Upload every file and folder in this directory to any static website
host. Configure the host to serve index.html as the default page.

Suitable hosts include Cloudflare Pages, Netlify, GitHub Pages, Amazon
S3 static hosting, and an ordinary company web server.

PROGRESS
Lesson and process progress is saved in each visitor's browser. It is
not shared across devices and does not identify the learner.

SOURCE
The content and assets were reconstructed from the supplied Rise 360
SCORM 1.2 export.
